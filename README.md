Simple 2D web-based platformer inspired by classic Mario-style games, set in a space environment.  
The player controls an astronaut who explores space-themed levels, collecting stars and fighting alien enemies.

## Game Concept
The objective of the game is to complete each level by reaching the end while collecting stars.  
Along the way, the player encounters different types of alien enemies. Aliens can be defeated by jumping on them.

## Gameplay
- 2D side-scrolling platformer
- Jump between platforms
- Collect stars scattered across the level
- Reach the finish point to complete the level

## Controls
- **Move Left:** Left Arrow
- **Move Right:** Right Arrow
- **Jump:** Up Arrow

## Theme
- Space and sci-fi setting
- Astronaut main character
- Multiple types of alien enemies
- Cosmic levels and backgrounds
